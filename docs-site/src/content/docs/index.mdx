## How each tool page is organized

Every tool page has the same anatomy so you can scan it quickly:

1. **Meta block** — version, status, Blender compatibility, download/source links.
2. **At a glance** — one-paragraph summary plus the most common day-to-day use case.
3. **Install** — drag-and-drop zip, or directory copy if applicable.
4. **UI tour** — screenshots of the panels with annotation notes.
5. **Day-to-day vs Advanced** — tabs separate the common path from edge-case features.
6. **Troubleshooting** — known gotchas and fixes.
7. **Changelog** — recent versions and what changed.

If you maintain these tools and want to refresh a page after a release, see [Updating these docs](/Toolings/guides/updating-docs/).
