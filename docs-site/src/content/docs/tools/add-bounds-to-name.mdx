## Summary

Rename selected objects to include their bounding-box dimensions: `wall_400x200x10cm`, `rock_2.5x1.8x2.1`. Choose units (m / cm / mm), rounding mode (none / round / floor / ceil) and increment, axis swizzle, separator, padding, and optional unit suffix. Save configurations as presets for studio-wide consistency.
