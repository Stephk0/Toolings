## Summary

Add a modifier to one object, broadcast it to many. Source modifier tagged with a 6-char hash ID (e.g. `Bevel (Source:abc123)`); targets get driver-bound copies that update live. Supports vanilla modifiers and Geometry Nodes inputs, with manual sync for non-drivable reference fields. Includes **Find Source** navigation to walk the driver chain back to the origin.
