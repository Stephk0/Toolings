## Summary

A targeted fork of the **ModifierList** addon. Upstream's list view dropped the GN Input Attribute toggle on Blender 5 due to an API change; this fork restores it. Otherwise identical to the original.

Use this if you rely on the toggle in list-view mode on Blender 5. On Blender 4.x the upstream addon is fine.
