┌──────────────────────────────────────────────────────────────────────────────┐
│ QUICK START                                                                  │
└──────────────────────────────────────────────────────────────────────────────┘

1. Open Blender 4.5+
2. Scripting Workspace → Open "quick_test_scene.py"
3. Run Script (▶ icon or Alt+P)
4. Adjust parameters in Modifier Properties panel
5. Switch to Geometry Nodes workspace to see nodes

┌──────────────────────────────────────────────────────────────────────────────┐
│ PARAMETERS                                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

BASE THICKNESS (0.01-2.0m, default: 0.1)
  ├─ 0.05-0.1  : Small trees, bushes
  ├─ 0.2-0.5   : Medium trees
  └─ 0.5-2.0   : Large/ancient trees

BRANCH LENGTH (0.1-10.0m, default: 1.0)
  ├─ 0.5-1.5   : Compact, bushy
  ├─ 1.5-3.0   : Medium spread
  └─ 3.0-5.0   : Wide, dramatic

ITERATIONS (0-10, default: 2)
  ├─ 0         : Trunk only
  ├─ 1         : Trunk + main branches
  ├─ 2-3       : Natural tree (recommended)
  ├─ 4-5       : Dense, detailed
  └─ 6+        : ⚠️  SLOW! Use with caution

RANDOM SEED (0-999999, default: 0)
  └─ Change for different variations (same settings, new shape)

ANGULAR SPREAD (0.0-1.0, default: 0.3)
  ├─ 0.0-0.2   : Very straight, formal
  ├─ 0.2-0.4   : Natural (RECOMMENDED)
  ├─ 0.4-0.6   : Wild, chaotic
  └─ 0.6-1.0   : Extreme chaos

┌──────────────────────────────────────────────────────────────────────────────┐
│ NODE FRAMES (Geometry Nodes Workspace)                                      │
└──────────────────────────────────────────────────────────────────────────────┘

INPUT PROCESSING (Green)
  └─ Converts input curve, resamples points

ATTRIBUTE INITIALIZATION (Blue)
  └─ Stores: iteration_level, branch_id, thickness, curve_parameter

BRANCH GENERATION (Red)
  └─ Repeat Zone: Creates new branch levels

GROWTH DIRECTION (Orange)
  └─ Calculates where branches grow (normal + random)

GEOMETRY BUILDER (Purple)
  └─ Converts curves to mesh with radius

┌──────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

NO BRANCHES?
  ✓ Iterations > 0
  ✓ Branch Length > 0
  ✓ Try Angular Spread = 0.5

BLENDER SLOW/FROZEN?
  ✓ Iterations = 2-3 max
  ✓ Disable viewport realtime update
  ✓ Close other apps

WEIRD SHAPES?
  ✓ Try different Random Seed
  ✓ Adjust Angular Spread
  ✓ Check input curve quality

CAN'T FIND MODIFIER?
  ✓ Modifier Properties (wrench icon)
  ✓ Named "ProceduralTreeGenerator"
  ✓ Make sure script completed

┌──────────────────────────────────────────────────────────────────────────────┐
│ PERFORMANCE GUIDELINES                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

VIEWPORT WORK:
  Iterations: 2-3  |  Smooth & Interactive

FINAL RENDER:
  Iterations: 3-5  |  Some lag OK

HIGH DETAIL:
  Iterations: 5-7  |  Slow but manageable

⚠️  EXTREME (NOT RECOMMENDED):
  Iterations: 8+   |  Very slow, high memory

┌──────────────────────────────────────────────────────────────────────────────┐
│ WORKFLOW EXAMPLES                                                            │
└──────────────────────────────────────────────────────────────────────────────┘

QUICK TREE:
  1. Run quick_test_scene.py
  2. Iterations → 3
  3. Random Seed → vary until satisfied
  4. Done!

FOREST:
  1. Create one tree
  2. Shift+D to duplicate
  3. Change Random Seed on duplicate
  4. Repeat for variations
  5. Arrange in scene

EXPORT FOR GAME:
  1. Create & adjust tree
  2. File → Export → FBX
  3. Use in game engine

┌──────────────────────────────────────────────────────────────────────────────┐
│ ATTRIBUTES CREATED                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

iteration_level   : Int    : Branch depth (0=trunk, 1=main, etc.)

┌──────────────────────────────────────────────────────────────────────────────┐
│ TIPS & TRICKS                                                                │
└──────────────────────────────────────────────────────────────────────────────┘

→ Start with defaults, adjust ONE parameter at a time
→ Random Seed is your best friend for variation
→ Save interesting seeds (write them down!)
→ Iterations 2-3 is the sweet spot for most trees
→ Higher Angular Spread = droopier/wilder
→ Lower Angular Spread = straighter/formal
→ Duplicate trees and vary seeds for forests
→ Use Collections to organize tree variations
→ Export to FBX for game engines
